# 残障支持 · 行业知识包

## 核心政策
- 《残疾人保障法》(2008修订)
- 《"十四五"残疾人保障和发展规划》
- 《无障碍环境建设法》(2023)
- 中国签署《残疾人权利公约》(2008)

## 行业术语
| 术语 | 定义 |
|------|------|
| 残疾人两项补贴 | 困难残疾人生活补贴+重度残疾人护理补贴 |
| 无障碍环境 | 便于残疾人通行的物理和信息环境 |
| 融合教育 | 残疾儿童与普通儿童一起接受教育 |
| 辅助性就业 | 为重度残疾人提供的庇护性就业形式 |
| 康复辅具 | 帮助残疾人恢复或补偿功能的器具 |
