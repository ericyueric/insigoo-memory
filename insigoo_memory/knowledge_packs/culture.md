## 行业术语
| 术语 | 定义 |
|------|------|
| 非遗 | 非物质文化遗产：口头传统、表演艺术、社会实践、仪式等 |
| 活态传承 | 在当代社会生活中持续实践和传承 |
| 传承人 | 被认定具有非物质文化遗产传承能力的人 |
| 文化生态保护区 | 以保护非遗为核心的文化空间 |
