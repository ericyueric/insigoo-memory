# 乡村振兴 · 行业知识包

## 核心政策
- 《乡村振兴促进法》(2021)
- 《"十四五"推进农业农村现代化规划》
- 每年中央一号文件（三农主题）
- 国家乡村振兴局（原国务院扶贫办）

## 行业术语
| 术语 | 定义 |
|------|------|
| 乡村振兴20字方针 | 产业兴旺、生态宜居、乡风文明、治理有效、生活富裕 |
| 三产融合 | 农业+农产品加工+乡村旅游服务业融合发展 |
| 驻村帮扶 | 选派干部到农村基层开展帮扶工作 |
| 合作社 | 农民专业合作社，农村集体经济组织形式 |
| 衔接资金 | 巩固拓展脱贫攻坚成果同乡村振兴有效衔接资金 |

## 常见资助方
- 中国扶贫基金会（现中国乡村发展基金会）
- 中国农业银行 - 乡村振兴基金
- 中国光大集团 - 乡村振兴公益基金

## 同类组织
- 中国扶贫基金会：乡村发展项目
- 黑土麦田：大学生返乡创业支持
- 阳光三农：农村信息服务
