## 行业标准与术语

| 术语 | 定义 |
|------|------|
| 义务教育巩固率 | 九年义务教育阶段在校生数与适龄人口的比例，国家目标95%+ |
| 师生比 | 教师数量与学生数量的比率，农村小学标准1:23 |
| STEAM教育 | 科学、技术、工程、艺术、数学融合教育模式 |
| 控辍保学 | 防止学生辍学并保障其完成义务教育的政策体系 |
| 随迁子女 | 跟随进城务工父母到城市生活的学龄儿童 |
| 留守儿童 | 父母一方或双方长期外出务工而留在农村的儿童 |

## 同行动态参考

- 美丽中国：支教老师驻村模式，每年派遣约300名项目老师
- 真爱梦想：建设"梦想中心"素养教育教室，覆盖全国31省
- 免费午餐：为乡村学校提供免费午餐，累计覆盖超1500所学校
- 蓝信封：留守儿童书信陪伴项目

## 常见项目逻辑框架

```
问题：乡村教育资源不均衡 → 
投入（教师培训、教具、互联网设备）→ 
活动（支教、远程课堂、教师工作坊）→ 
产出（开课数、培训人次）→ 
成果（学习成绩提升、辍学率下降、学校满意度提升）
```
