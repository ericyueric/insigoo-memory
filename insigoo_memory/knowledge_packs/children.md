## 行业术语
| 术语 | 定义 |
|------|------|
| 困境儿童 | 因家庭经济、身体等原因处于困难境地的儿童 |
| 事实无人抚养儿童 | 父母双方不能正常履行抚养义务的儿童 |
| 儿童主任 | 村/社区指定的儿童保护负责人 |
| 早期发展 | 针对0-3岁儿童的综合发展干预 |
| 替代性养护 | 对无法在原生家庭生活的儿童提供其他养护方式 |
