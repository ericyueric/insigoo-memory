## 行业术语
| 术语 | 定义 |
|------|------|
| 社区治理 | 多元主体参与社区公共事务的合作治理 |
| 社区营造 | 居民共同参与改善社区生活环境的持续性过程 |
| 社区基金会 | 以社区为基础募集和管理慈善资源的组织 |
| 四社联动 | 社区、社会组织、社工、志愿者的协作模式 |
| 社区微更新 | 小规模渐进式改善社区公共空间 |
