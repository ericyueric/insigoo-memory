## 行业术语
| 术语 | 定义 |
|------|------|
| 基本医疗保险 | 覆盖城镇职工和城乡居民的基本医疗保障体系 |
| 大病救助 | 针对重大疾病患者的经济援助体系 |
| 健康扶贫 | 通过医疗保障防止因病致贫、因病返贫 |
| 分级诊疗 | 基层首诊、双向转诊、急慢分治、上下联动 |
| 罕见病 | 患病率低于1/10000的疾病，中国确认121种 |
