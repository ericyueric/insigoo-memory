## 行业术语
| 术语 | 定义 |
|------|------|
| 应急响应级别 | I级(特别重大)到IV级(一般)四级响应 |
| 备灾仓库 | 预先储备救灾物资的仓库网络 |
| 韧性社区 | 具备抵御、适应和恢复能力的社区 |
| PPP框架 | 预测、预防、准备三阶段灾害管理框架 |
| 人道主义标准 | 受灾人群最低生活保障标准(Sphere标准) |
