# 志愿服务 · 行业知识包

## 核心政策
- 《志愿服务条例》(2017)
- 《关于推进志愿服务制度化的意见》
- 中国志愿服务联合会成立(2013)
- 《关于健全新时代志愿服务体系的意见》(2023)

## 行业术语
| 术语 | 定义 |
|------|------|
| 志愿时长 | 志愿者服务时间的记录单位 |
| 志愿团体 | 依法成立的组织或单位内部成立的志愿服务队伍 |
| 志愿中国 | 全国志愿服务信息系统，记录志愿服务时长 |
| 志愿保险 | 为志愿者在服务期间提供的人身意外保险 |
| 支援协议 | 志愿服务组织与志愿者之间的书面约定 |
