# 妇女发展 · 行业知识包

## 核心政策
- 《妇女权益保障法》(2022修订)
- 《中国妇女发展纲要(2021-2030)》
- 《反家庭暴力法》(2016)
- 北京宣言和行动纲领(1995)后续行动

## 行业术语
| 术语 | 定义 |
|------|------|
| 性别主流化 | 在所有政策中纳入性别平等视角 |
| 妇女赋权 | 增强妇女在经济、社会、政治领域的自主性 |
| 性别预算 | 从性别角度分析公共预算分配 |
| 反家暴 | 预防和制止家庭暴力的法律和社会干预体系 |
| 留守妇女 | 因丈夫外出务工而留守农村的已婚妇女 |
