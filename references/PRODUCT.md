# insigoo-memory — 不会丢文件的组织

> 你只管往里放文件，剩下的它来。

---

## 它是什么

你们机构的电脑里，有没有这种情况？

- "上次那个项目的活动照片放哪了？"
- "资助方的合同快到期了我才想起来"
- "换了个人接手，之前做过的方案全找不到了"
- "每天打开十几个文件夹，不知道哪些重要哪些可以删"

**insigoo-memory 就是来解决这个的。**

它是一个软件助手，装在你电脑上。你把你平时工作的文件夹告诉它，它会自动：

1. **帮你的文件分类** — 不用手动建目录，它读得懂文件里写的是什么
2. **提醒你该做的事** — 从文件里发现"下个月要交报告""项目要启动了"，催你排上日程
3. **告诉你缺什么** — "99公益日快到了，你怎么还没有准备海报？"
4. **做个自动目录** — 新同事来了打开就能看到：这个组织做过什么、正在做什么、准备做什么

---

## 它能做什么

你丢进去的东西，会被自动分到 **9 个格子**里：

| | 放什么 |
|------|------|
| 📰 **行业资讯** | 政府发的政策文件、看到的资助机会、同行在做什么 |
| 📚 **研究学习** | 你看过的行业报告、学过的课程、好的方法论 |
| 🎨 **设计物料** | 海报、Logo、公众号素材、宣传品 |
| 📝 **项目方案** | 项目计划书、申请表、预算 |
| 🏃 **项目痕迹** | 活动照片、签到表、志愿者名单、受益人档案 |
| 💰 **财务资料** | 预算执行表、捐赠记录、审计报告 |
| 📊 **监测评估** | 项目数据、满意度调查、评估报告 |
| 📦 **结项资料** | 结项报告、成果总结、新闻稿 |
| 🏢 **行政人事** | 章程、理事会纪要、员工手册、年检材料 |

**你不需要手动建这些目录。** 丢一个文件进去，它就知道该放哪。

## 怎么开始

### 第一步：安装

如果你用的是 Windows 电脑：

```
❶ 打开「命令提示符」或者「PowerShell」
❷ 输入：pip install insigoo-memory
❸ 回车，等它装完
```

如果你不太懂这个——找一个懂电脑的同事帮你装，就这一行命令。

> 💡 免费。不需要买服务器，不需要注册账号。所有数据都在你自己电脑上。

### 第二步：告诉它你的文件夹在哪

```
insigoo-memory init -w "D:\我的机构文件" -i 环保 教育
```

这行命令的意思是：扫描 `D:\我的机构文件` 这个文件夹，告诉它你的机构主要做环保和教育。

### 第三步：看它给你生成的看板

```
insigoo-memory dashboard
```

浏览器会自动打开一个页面，像这样：


## 回答你可能会问的

**Q: 我的文件很乱，什么格式都有，它能行吗？**
能。Word、Excel、PDF、照片、扫描件、微信聊天记录导出的文本——都可以。

**Q: 它会把我文件弄坏吗？**
不会。它只读你的文件，不改动原文件。所有分类信息存在一个单独的文件夹里。

**Q: 数据安全吗？**
所有数据都在你自己的电脑上，不联网，不上传。你的捐赠人名单、财务数据、项目方案不会离开你的电脑。

**Q: 能不能多人用？**
现在是一台电脑一个账号。以后会支持团队版。

**Q: 贵吗？**
免费。这是因思阁（insigoo）为公益组织做的工具。
